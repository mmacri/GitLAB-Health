    What counts as first value

    First value is reached when at least 10% of production licenses are activated.

    • SaaS Auto-derived when known license utilization reaches 10% and is recorded in the system.
    • Self-managed CSM updates the first value date using the best estimate from usage evidence.
    • Evidence GitLab installed, user management enabled, and active usage in production projects.
